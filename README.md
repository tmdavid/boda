# Sofia i David - Wedding Website

A bilingual (Catalan/Spanish) wedding website.

## Development

```bash
npm install
npm start
```

The site will be available at http://localhost:5000

## Deployment

Deploy to Netlify:

```bash
netlify deploy --prod
```

## Configuration

Edit `src/config.json` to update:
- Couple names
- Wedding date
- Venue details
- Schedule

Edit `src/helpers/translations.js` to update text content in both languages.
